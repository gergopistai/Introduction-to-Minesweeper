Introduction

This is a simple, static website made for a small university project, it will explain the basics of minesweeper and show some patterns.

In my opinion one of the best places to play minesweeper is at minesweeper.online.

Basics of Minesweeper

How the game works

A game of minesweeper is played on a grid of cells. Every cell can be opened, but the point of the game is to find (and flag) all the cells that are mines. If you click on a mine you lose the game. The game can be only be won if every non-mine cell is opened. You can open cells by left clicking on them.

Most opened cells will have a number on them. This number represents how many mines are adjacent to that given cell (adjacency here includes diagonal cells as well). This also means that open cells with no numbers have no mines around them.

There are usually two game modes: Normal and No Guess (NG for short).

Normal mode is entirely randomly generated, so there are usually some guessing whether a certain cell is a mine or not. Situations where you can not tell which one of two given cells is a mine are also called 50-50.

NG games are randomly generated as well, but in a way that they can be won without guessing, only by recognizing patterns.

You can right click on an unopened cell to flag them. The flag is used to indicate that there is a mine there. It also prevents you from accidentally opening the given cell.

Tips for beginners:

  • Try to start every game from the center.
  • Leave guessing only as a last solution.
  • If a cell number is equal to the unopened cells around that cell, then all of them are mines.
  • If a cell number is equal to the number of flags around it, then every other adjacent cell is safe.
  • The amount of mines left can sometimes help determine which of the last few cells are mines.
  • You can improve by learning patterns.

Some basic patterns

Hover over individual cells to see more information about them.

1-1-X Pattern

Because of the leftmost 1, there is a mine guaranteed to be on either the first or second cell. In either case, the second 1 is satisfied, which means there can't be a mine on the third unopened cell.

"First cell" - Might be a mine.
"Second cell" - Might be a mine.
"Third cell" - Guaranteed to be safe.
1 "Leftmost 1"
1 "Second 1"
1 1 1

1-2-1 Pattern

The leftmost 1 guarantees that there is a single mine in one of the two cells above it. It doesn't matter which one is the mine, since it will satisfy one mine for the 2, therefore the 2's other mine must be at the thrid position. Then the second 1 will be satisfied. Which means that the second cell is empty (since there can't be another mine adjacent to the second 1), therefore the first cell is also a mine.
